# AI 编程学习作业

本仓库是 AI 编程学习课程的作业记录，包含 Markdown 语法学习和 Git/GitHub 使用实践。

## 目录结构

```
.
├── 001-学习markdown/     # Markdown 语法学习作业
└── 002-git_github学习/   # Git 与 GitHub 使用学习作业
```

## 作业列表

### 001 - 学习 Markdown

- 学习 Markdown 基本语法与格式
- 参考网站：https://sjjliqpl.github.io/ai-coding-learning/#/markdown
- 对照渲染效果，使用 Markdown 编辑器完成文档编写

### 002 - Git & GitHub 学习

- 学习 Git 和 GitHub 的基本概念
- 参考网站：https://sjjliqpl.github.io/ai-coding-learning/#/git
- 在本地创建 Git 仓库并提交修改
- 在 GitHub 上创建公开仓库并关联本地仓库

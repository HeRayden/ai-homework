## 二、作业
1. 通过学习，把[[001-学习markdown]](../001-学习markdown)的作业在本地创建自己的git仓库，并提交至少一次修改。
    > 可以使用 [webstorm](https://www.jetbrains.com/webstorm/) 或者 [vscode](https://code.visualstudio.com/) 等工具来创建git仓库并提交修改。
2. 使用gmail注册github账号。
3. 在github上创建一个新的仓库，命名为markdown-homework，并把本地的git仓库关联到github上的这个新仓库。

## 三、作业验收标准
 
- [ ] github账号，只提供名称即可
- [ ] markdown-homework 仓库的链接。
